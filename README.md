🫀 Computational Identification of Differentially Expressed Genes and Pathways Associated with Hypertrophic Cardiomyopathy Using R

This project performs a comprehensive transcriptomic analysis of Hypertrophic Cardiomyopathy (HCM) using publicly available RNA-seq data from GEO (GSE180313). Differential gene expression analysis was carried out using DESeq2, followed by KEGG pathway enrichment analysis using ClusterProfiler to identify key molecular pathways involved in HCM.

📌 Objectives

Retrieve RNA-seq data from GEO (GSE180313)
Perform quality-controlled differential expression analysis
Identify significantly upregulated and downregulated genes
Conduct KEGG pathway enrichment analysis
Visualize results using Volcano Plot, Heatmap, MA Plot, and Bar Plot
Interpret biological pathways associated with Hypertrophic Cardiomyopathy
